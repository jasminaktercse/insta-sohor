What is Js event loop?

The event-loop is an infinite loop that in each iteration verifies in the event queue if some event was triggered.and The event loop is the key of asynchronous programming in JavaScript. It just has one job: to keep an eye on the Call Stack and the Callback Queue. If the Call Stack is empty, the first event in the queue will be pushed to the Call Stack, which will effectively run it. Each event is nothing more than a function callback.
